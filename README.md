# Health Care
Hi! This is a web application for health care.
Build with `NextJS 13`.

**Requirement:**
Minimum `NodeJS` version `14.18.0`

## How to
###	Install
    yarn
or

    yarn install
### Run development (debug)
    yarn dev
### Build production (release)
    yarn build
### Run production (release) (only run after build)
    yarn start
